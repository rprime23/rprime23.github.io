Archery Arena

IN PROGRESS

Archery Arena is a VR experience that creates a challenging 360 degree experience with varying styles of targets. The user attempts to hit as many targets as possible in a set amount of time. Can you earn the high score?

Archery Arena is a solo project. This project was built in Unreal Engine 5 using Unreal blueprinting. I used the VR template, but did all the modeling and blueprinting.
