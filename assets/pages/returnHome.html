Return Home

Return Home is a VR imersive experience that focuses on encouraging a return to nature. It uses effects to help create as immersive of an experience as possible. It has eye adaptation, a custom written bloom script, fog, a day night cycle, and uses tree and grass shaders to create the environment.

Return Home was created in a four person team. This project was built in Unity using C# scripting. I contributed to the bloom shader, the bloom script manipulation scripts, and the the basic assembly of the environment.
